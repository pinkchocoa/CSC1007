This function changes the values, using the preset test case to runProgram() to proceed with the banker algorithm program.

Note
The parameters are all constant type in order to make its value fix, to prevent changes in the underlying value.
Parameters
psizeThis is refering to the int type variable that represent the process size.
rsizeThis is refering to the int type variable that represent the resource size.
avThis is a pointer to an integer to dereference available value.
alThis is a pointer to an integer to dereference allocation value.
mThis is a pointer to an integer to dereference max value.

These are the 6 preset testcases: testCase1() testCase2() testCase3() testCase4() testCase5() testCase6()
